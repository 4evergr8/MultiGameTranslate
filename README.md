# MultiGameTranslate 🎮✨

一个**多引擎游戏翻译补丁生成器** 🇯🇵➡️🇨🇳，帮助玩家和翻译者快速为视觉小说/ADV游戏生成中文翻译补丁！

目前项目处于**测试阶段** 🔧，**仅完整支持 TyranoScript 引擎**（TyranoBuilder / TyranoScript 游戏），未来计划支持 RPG Maker 等更多引擎 🚀。

本工具基于 Mtool 生成的 `ManualTransFile.json` 进行翻译管理，支持旧版本翻译自动同步，大幅减少重复工作量 😊。

## 特性 🌟

- 支持 TyranoScript 引擎翻译补丁生成 ✅
- 旧版本翻译自动同步（避免重复翻译相同文本） 🔄
- 预处理自动过滤无关内容，并按日语占比排序（优先翻译日语多的条目） 📊
- 简单易用的手动翻译界面 ✍️
- 一键生成补丁，直接替换游戏文件 🎉
- 未来计划：支持 RPG Maker 等更多引擎 🛠️

## 使用方法 📝

请先使用 **Mtool** 提取游戏文本，生成 `ManualTransFile.json` 并放入项目根目录。

按顺序运行以下四个脚本：

1. **1,预处理.py** 🧹  
   清理无关内容（如代码、重复项），并按日语占比从高到低排序。  
   运行后生成优化后的 `ManualTransFile.json`，方便优先翻译核心内容。

2. **2,同步旧版本.py** （可选，强烈推荐） 🔁  
   若有旧版本翻译文件（同样为 `ManualTransFile.json`），运行此脚本可自动同步已翻译内容。  
   请将旧文件命名为 `old_ManualTransFile.json` 放入根目录。

3. **3,翻译.py** ✏️  
   启动手动翻译界面，逐条显示原文与翻译框。  
   支持随时保存进度，中途可中断继续。

4. **4,替换.py** （当前为 `4,替换Tyrano.py`） 🔨  
   使用翻译完成的 json 生成 Tyrano 专用补丁。  
   输出文件夹中生成的替换文件，直接覆盖到游戏目录即可生效！

### 注意事项 ⚠️

- 当前仅支持 Tyrano 引擎，其他引擎替换脚本开发中 ⏳
- 应用补丁前请务必备份游戏原文件 🛡️
- 项目仍在测试，如遇 bug 欢迎提交 Issue 🐛
